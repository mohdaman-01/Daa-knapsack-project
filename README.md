# Stock Investment Optimizer

Application of Knapsack Algorithm in Stock Investment Optimization

## Features
- Interactive stock portfolio optimization
- 0/1 Knapsack algorithm implementation
- Real-time budget constraint handling
- Visual portfolio representation

## Deploy to Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy)

### Manual Deployment
1. Drag and drop this folder to https://app.netlify.com/drop
2. Your site will be live instantly!

## Local Development
Simply open `index.html` in your browser. No build process required.

## How It Works
The application uses dynamic programming to solve the 0/1 Knapsack problem, maximizing expected returns while staying within your investment budget.
